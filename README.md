## 📌 Sobre o Projeto

Este é um projeto acadêmico desenvolvido para a disciplina de **Web Development** da [FIAP](https://www.fiap.com.br/) — **Global Solution 2026**.

O **OrbitAlert** é uma solução que combina dados de satélites (NASA FIRMS, INPE, Copernicus) com hardware de baixo custo para enviar alertas físicos e sonoros de enchentes e queimadas a comunidades isoladas do Brasil, operando totalmente sem internet, celular ou infraestrutura de telecomunicações.

Este repositório contém o **frontend** do projeto — um site institucional responsivo que apresenta o problema, a tecnologia, os objetivos e os benefícios da solução.

---

## 🧰 Tecnologias Utilizadas

- HTML5 semântico
- CSS3 (variáveis customizadas, Flexbox, Grid, temas claro/escuro)
- JavaScript vanilla (slideshow, lightbox, quiz interativo, validação de formulário)
- [Boxicons](https://boxicons.com/) para ícones

## ✨ Funcionalidades

- **Tema claro / escuro / sistema** — alternância com persistência
- **Slideshow** com controles manuais e indicadores
